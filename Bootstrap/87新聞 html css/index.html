【社論】蔡英文急什麼：兩岸不通，電力不夠

蔡英文政府最近有一個大動作，一個小動作，十分惹人注目，分別是深澳電廠環評強度關山，以及不斷放話質疑「馬習會」有檔案沒移交。 這兩者都姿勢難看，輿論反應不佳，但仍不計毀譽為之，反暴露出蔡英文總統的急切感：兩岸持續不通，電力未來不夠，恐成為大選蔡政府能否得到人民支持的重大變數。
